# logic-calculator
## 项目背景
在师傅排名和晋级考核中，如何公平、科学地确定晋级标准是一个关键问题。传统的考核方法往往只关注单一维度的表现，而忽略了师傅在多个维度上的综合表现。为了解决这一问题，我们开发了逻辑计算器，专门用于计算短板阈值，确保最终入选的师傅是那些在所有维度上都相对均衡且优秀的群体。

## 项目功能
- 核心计算 ：根据参与排名师傅数 (N) 和晋级目标比例 (P)，计算短板阈值 (T)
- 维度分析 ：支持自定义考核维度数 (d)，从1到多个维度进行分析
- 相关性考虑 ：引入相关性系数 (ρ)，反映不同考核维度之间的相关性
- 多场景假设 ：提供独立假设和正相关假设下的计算结果，以及基于相关性系数的综合结果
- 直观输出 ：将计算结果转换为清晰的晋级分数线说明，如"取分数从高到低的第X名的师傅分数作为晋级分数"
- 案例数据 ：自动生成不同晋级比例下的案例数据表，便于对比分析
- 模拟数据 ：提供随机生成模拟数据的功能，方便快速测试不同场景
## 技术实现
- 前端技术 ：HTML5 + CSS3 + JavaScript，响应式设计，界面友好
- 计算逻辑 ：基于概率论和统计学原理，实现多种假设下的短板阈值计算
- 部署方式 ：通过GitHub Pages部署，支持互联网访问和分享
## 应用场景
- 师傅晋级筛选 ：为企业提供科学的晋级标准，确保筛选出综合能力强的师傅
- 绩效考核优化 ：帮助企业分析不同考核维度的重要性和相关性
- 人力资源决策 ：为人力资源管理提供数据支持，优化人才选拔机制
- 行业标准制定 ：为行业内的师傅评价提供参考标准
## 项目价值
- 科学性 ：基于概率论和统计学原理，提供客观、准确的计算结果
- 灵活性 ：支持多种参数设置，适应不同企业的考核需求
- 易用性 ：界面简洁直观，操作简单，无需专业知识即可使用
- 可扩展性 ：代码结构清晰，便于后续功能扩展和维护
通过逻辑计算器，企业可以更加公平、科学地进行师傅排名和晋级筛选，确保选拔出真正优秀的人才，同时为师傅提供明确的努力方向，促进整体服务质量的提升。
